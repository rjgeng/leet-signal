**Example**

Input:
```
n == 3
```
-   Output:
```
true
```
Input:
```
n == 1
```
-   Output:
```
false
```

**Input/Output**

* **[execution time limit] 4 seconds (js)**

* **[input] integer n**

    A number to test.

    *Constraints*

    -100 <= n <= 100

* **[output] boolean**

    True if the number is between 2 and 7, exclusive.
