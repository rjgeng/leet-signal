## Description
------
Return a reversed version of a string.

If the string is empty, return an empty string.

**Example**
Input:
```
s == "abcdef"
```
-   Output:
```
"fedcba"
```
Input:
```
s == ""
```
-   Output:
```
""
```

**Input/Output**

* **[execution time limit] 4 seconds (js)**

* **[input] string s**

    The input string.

* **[output] string**

    The reversed version of the string.
