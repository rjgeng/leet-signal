```
a == [0, 5, 6, 2, 7, 8, 5, 3, 4, 5]
x = 5     Remove all 5s
```

```
[0, 6, 2, 7, 8, 3, 4]
```

    The value to delete.

    *Constraints*

        1 <= a.length <= 100
